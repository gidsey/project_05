# Treehouse Python Techdegree: Project 05

## Personal Learning Journal with Flask

### Description
A personal journal application that allows users to create an account, login and post journal entries. Each journal entry can also be tagged with relevant keywords.


### Installation and use
This project has been built using a Pyton Virtualenv and the dependencies for this are set out in the requirements.txt file included in this repo.

To run the app, launch 'app.py' in the shell and access the site at http://127.0.0.1:5000/ in your browser.

Please note the following
* Anonymous users can view all entries.
* Users need to be logged in order to post a new entry.
* Testing credentials are listed below, alternatively new accounts can be created by following the 'Sign up' link.
* Users can only edit or delete their own entries.
